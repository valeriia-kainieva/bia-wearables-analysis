BIVA & Wearables Data Analysis

This project explores physiological data obtained from wearable devices and bioelectrical impedance analysis (BIA).

The analysis focuses on relationships between variables such as:

diving depth
cardiovascular endurance
muscle mass

Data visualization is performed using:

matplotlib
seaborn
Plotly

Key Features
Correlation analysis (e.g., depth vs cardiovascular endurance)
Trend line visualization
Interactive plots (Plotly)
Small dataset exploratory analysis

Example Insight

A negative correlation was observed between diving depth and cardiovascular endurance; however, the dataset is small (n = 4), so conclusions are limited.

Technologies
Python
Pandas
Matplotlib
